# Iris Flower Classification using KNN

This project uses the K-Nearest Neighbors (KNN) algorithm to classify Iris flowers.

## Dataset
The dataset used is the built-in Iris dataset from Scikit-learn.

## Libraries Used
- pandas
- scikit-learn

## Features
- Load Iris dataset
- Convert dataset into DataFrame
- Train KNN model
- Evaluate accuracy
- Display classification report
- Display confusion matrix
- Predict new flower species

## How to Run

```bash
python iris_classifier.py
